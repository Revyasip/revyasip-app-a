- 🎯 Easy table booking system
- 💫 Beautiful animations and transitions

### 🚀 Quick Start

1. Clone the repository
2. Open in your favorite editor
3. Start the server:
```bash
python -m http.server 5000
```
4. Visit `http://localhost:5000` to view the app
